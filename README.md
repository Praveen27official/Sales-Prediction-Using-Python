# 📊 Sales Prediction using Python

This project predicts product sales based on advertising budgets spent on TV, Radio, and Newspaper platforms using Machine Learning.

---

## 🚀 Project Overview

Sales prediction is an important task for businesses to estimate future product demand and optimize advertising strategies.

In this project:
- Data analysis is performed on advertising datasets
- Machine Learning is used to predict sales
- Linear Regression is implemented for prediction

---

## 📁 Dataset

The dataset contains:
- TV advertising budget
- Radio advertising budget
- Newspaper advertising budget
- Sales
